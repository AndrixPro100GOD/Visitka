# Памятка по проекту Visitka

## База проекта
- Проект полностью статический: `index.html` + `index.css` + папка `images/`.
- Сборка не нужна. Сайт должен открываться напрямую как `file://.../index.html`.
- Для GitHub Pages главная страница — `index.html` в корне репозитория.

## Контент и структура
- Тексты и блоки редактируются в `index.html`.
- Стили и анимации редактируются в `index.css`.
- Использовать изображения только из `images/`.
- Внутренние переходы — только якоря (`#projects`, `#materials`, `#contacts`).

## Фон и визуал
- Общий фон страницы: `images/Background.png` через CSS.
- Между секциями использовать мягкий fade по прозрачности, чтобы скрывать неровности текстуры.
- Не делать фон слишком контрастным, чтобы текст оставался читаемым.

## Кэш и обновления (важно)
- При каждом релизе менять версию в query-параметрах:
  - `index.html`: `index.css?v=...`
  - `index.css`: `Background.png?v=...`
- Рекомендуемый формат: `v=YYYYMMDD-N` (пример: `v=20260218-3`).
- После деплоя проверить сайт в режиме инкогнито и обычным `Ctrl+F5`.

- [ ] Мобильная версия не ломает сетки.
- [ ] Версии для cache-busting обновлены.
