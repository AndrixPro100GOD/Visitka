# КУБ-СТРОЙ — статичный сайт-визитка

Проект работает без сборки: открывайте `index.html` напрямую или публикуйте репозиторий на GitHub Pages.

## Локальный запуск

- Вариант 1: открыть `index.html` в браузере.
- Вариант 2 (рекомендуется): запустить любой статический сервер в корне проекта.

## Публикация на GitHub Pages

1. Загрузить проект в репозиторий.
2. В настройках репозитория открыть **Pages**.
3. Выбрать источник: ветка `main` (или `master`) / папка `/ (root)`.
4. Убедиться, что главная страница — `index.html`.
